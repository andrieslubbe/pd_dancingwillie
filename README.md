
## Run using browser
1. Go to https://play.date/pulp/ and sign in
2. Download `Willie Dancer.json`
3. Import into pulp
4. Press <ctrl> + <b> to run the game

## Run using playdate
1. Download `Willie-Dancer.zip`
2. Extract zip
3. Upload `.pdx` file to Playdate
    - see https://help.play.date/games/sideloading/
